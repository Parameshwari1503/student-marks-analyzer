# Student Marks Analyzer

This project is built using Python and NumPy to analyze student marks efficiently.

## Features
- Calculates average marks
- Finds highest and lowest scores
- Assigns grades (A, B, C)
- Filters pass and fail students

## Technologies Used
- Python
- NumPy

## Purpose
This project demonstrates basic data analysis using NumPy arrays and vectorized operations
